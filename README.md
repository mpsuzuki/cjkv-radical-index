# CJKV 古字書 部首インデックス

## これは?

個人的なプロジェクトのために必要としている幾つかの古字書に対して整備した部首インデックスです。
編者個人の必要性から生まれたデータであり、フォーマットは大きく変更されることがあります。

## データファイル

全体としては機械読み取りに適したタブ区切りのデータファイルですが、
詳細なフォーマットについては各ファイルを参照してください。

*	玉篇 / 大広益会玉篇 (澤存堂本)  
	radicals_YP.txt
*	龍龕手鑑 (宋本)  
	radicals_LJSJ.txt
*	五音篇海 (成化丁亥重刊改併五音類聚四聲篇 / 至元重刊五音篇)  
	radicals_WYPH.txt
*	群籍玉篇 (新修絫音引證群籍玉篇)  
	radicals_QJYP.txt
